User Authentication Module
This project is an OS-based login system that allows secure registration and authentication of users. 
It includes password handling, user validation, and security question-based verification to protect access.

Features
New user registration
Login authentication
Password validation system
Security question verification
Limited login attempts for securite

How to Run
1. Open terminal in project folder
2. Compile the file:
gcc RaaZ.c -o RaaZ
3. Run the program:
./RaaZ
Purpose
This project demonstrates basic operating system-level authentication and access control.
It can be used for learning login mechanisms, input validation, and user security implementation.
